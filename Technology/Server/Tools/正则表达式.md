# 正则表达式例子
## 常用通配符说明
1. *：代表匹配任意字符
2. []: 代表“[”和“]”之间的某一个字符，比如[0-9]可以代表0-9之间的任意一个数字，[a-zA-Z]可以代表a-z和A-Z之间的任意一个字母，字母区分大小写。
3. ^: 代表结果取反，注意这个通配符必须要在[]中使用
3. 

### 例子
1. 查看当前目录下所有不是以.nm!结尾的文件
```
    查看文件名称不是以 .nm!结尾的，注意这里由于！是正则的符号所以需要转义。
    ls -l *[^.nm\!]
    查找以nm!结尾的只需要去掉^即可
    ls -l *.nm\!
```
